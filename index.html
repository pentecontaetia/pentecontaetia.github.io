HTML Semantic tags Reference

Semantic tags

Tag Name Description
<header> Header Container for introductory content or a set of navigational links
<nav> Navigation Defines a set of navigation links
<section> Section Defines a section in a document with the same theme
<article> Article Part that makes sense on it's own
<aside> Aside To give additional information but is not essential for understanding
<footer> Footer Content at the bottom of a page

Non-Semantic tags

Tag Name Description
<div> Division Defines a division or a section in HTML document
<span> Span An inline container marking part of a text or a document
